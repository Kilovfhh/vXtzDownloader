# vXtzDownloader

**A sleek Python GUI application to download YouTube videos and audio effortlessly!**

![vXtzDownloader Screenshot](assets/screenshot.png)  <!-- Add your screenshot here -->

---

## Features

- Clean and modern CustomTkinter-based GUI  
- Supports multiple formats: MP3, 720p, 1080p, Best Quality  
- Download progress with speed and ETA display  
- Download history saved and loadable  
- Splash screen with loading animation  
- Mini-game for fun during downloads  
- Open output folder button for convenience
